# Support

## Getting Help

Use [GitHub Discussions](https://github.com/steerlabs/opensteer/discussions) for:

- usage questions
- ideas and design discussions
- showcase and feedback

Use [GitHub issues](https://github.com/steerlabs/opensteer/issues) for:

- bug reports
- feature requests

When filing an issue, include:

- Opensteer version
- Node.js version
- OS/runtime details
- minimal reproduction
- expected vs actual behavior

## Security and Sensitive Reports

For vulnerabilities or sensitive disclosures, use the private channel in
[SECURITY.md](SECURITY.md) instead of public issues.
