# @opensteer/browser-core

Engine-neutral primitives and contracts for Opensteer browser backends.

Install this package directly only when you are implementing or testing a browser
engine against the Opensteer interfaces. Most users should install
[`opensteer`](../opensteer/README.md) instead.

```bash
pnpm add @opensteer/browser-core
```

The package exposes shared capability contracts, browser identity and geometry
types, network and snapshot helpers, and the conformance surface used by the
Opensteer engine packages.
