# @opensteer/engine-playwright

Playwright-backed browser engine for Opensteer.

Most users should install [`opensteer`](../opensteer/README.md) and let it pull
this package in automatically. Install this package directly only when you are
building custom runtimes on top of the lower-level engine surface.

```bash
pnpm add @opensteer/engine-playwright playwright
pnpm exec playwright install chromium
```
